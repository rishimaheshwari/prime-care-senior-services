IMAGES FOLDER
=============

You don't need any images for the website to work — it's designed to look
great without photos. But if you'd like to add some later, here's how.

------------------------------------------------------------
1. FOUNDER PHOTO (Dr. Maheshwari)
------------------------------------------------------------
- Save a headshot in this folder named exactly:  dr-maheshwari.jpg
  (a square image works best, e.g. 600 x 600 pixels)

- Then open index.html and founder.html and find the line that says:
      <span class="initials" aria-hidden="true">SM</span>
  Replace that single line with:
      <img src="images/dr-maheshwari.jpg" alt="Dr. Sanjay Maheshwari" />

  (There's a comment right above each spot telling you exactly where.)

------------------------------------------------------------
2. SOCIAL SHARE PREVIEW IMAGE (optional)
------------------------------------------------------------
- This is the picture that appears when the website link is shared in a
  text message or on Facebook.
- Save an image here named:  social-preview.jpg  (1200 x 630 pixels is ideal)
- It's already wired up in index.html (the og:image tag).

------------------------------------------------------------
TIPS
------------------------------------------------------------
- Use .jpg for photos and .png for logos/graphics.
- Keep files under ~500 KB so the site loads fast. You can shrink images
  for free at https://squoosh.app
